# Belgian_Dutch_FIR_flightmap_EBBU_EHAA

Ce dépôt contient :
- Un fichier GeoJSON avec les limites FIR Belges et Néerlandaises (EBBU, EHAA).

This repository contains:
- A GeoJSON file with Belgian and Dutch FIR boundaries (EBBU, EHAA).
